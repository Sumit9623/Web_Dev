there are 3 ways to CSS to the HTML markup :

  1. Inline
  2. Internal
  3. External

  1. Inline

    2. In this method we add CSS in HTML at perticular line. this is not good programing practice because adding CSS at every line not good habit. it mixes HTML and CSS . By using style attribute we can add inline CSS.

  2. Internal

    3. In this method we add all CSS in head tag it separate markup and CSS but in same file this also not good programing practice. by using style tag we can add CSS in same file.

  3. External

    4. In this method we make separate file For CSS and include it in HTML file. by using link we can add CSS file to the HTML
